## 🧠 About

This repository hosts the official **Gangsworld** landing page – a hacker‑style portfolio showcasing every project in the Gangsworld ecosystem.  
Each tool is designed to solve a real problem faced by content creators: from AI‑powered video clipping to brand deal management, audience analytics, and open‑source alternatives to expensive SaaS products.

The page features a **live CLI emulator** for our flagship project, **[ViralClip Swarm](https://github.com/Gangsworld3/viralclip-swarm)** , letting visitors experience the tool directly in their browser.

---

## ✨ Featured Project: ViralClip Swarm

**Turn 2‑hour streams into 10 viral shorts – automatically.**  
ViralClip Swarm uses AI (WhisperX, OpenCV, FFmpeg) to detect engagement peaks (laughter, loudness, chat activity) and exports vertical‑ready clips with dynamic captions.

- 🔥 **Active development** – [Star the repo](https://github.com/Gangsworld3/viralclip-swarm)
- 🖥️ **Live demo** – Try the CLI emulator on the [website](https://gangsworld3.github.io)

## 🚀 Live Demo

The website includes an **interactive terminal** that simulates the ViralClip Swarm CLI.  
Try commands like:

--num-clips 8
--analyze
help
clear

No installation required – just open the page and start typing.
